§ 02

Three messages.
No wallet. No keys. No app.

Natural-language intent in, confirmed on-chain transfer out. Under ten seconds.

01
Open the chat

§ 03

What each claw
plugs into.

StellarClaw is not a wallet. It's an agent on top of Stellar's programmable-money stack — six layers, each already shipped or shipping.

L1
Stellar + Soroban
Execution layer
WASM smart contracts with SDKs, CLI, RPC server, and a local sandbox. Multi-core scaling and an optimised fee model — the substrate every claw agent executes against. Ledger closes in ~5 seconds.
Substrate
L2
Soroban → MCP bridge
Agent control · DoraHacks
One CLI turns any Soroban contract into an MCP server. Agents invoke contract functions from natural prompts, with policy signers so the agent never holds full account control.
Policy signers
L3
x402 agentic payments
Settlement + guardrails
Fast, cheap stablecoin settlement. Next: x402-MCP — agents discover paid resources and chain API calls under user-defined spending policies. Soroban adds programmable limits, approvals, compliance.
Spending policies
L4
Real-world off-ramps
Fiat surface
MoneyGram across 170+ countries and 470,000 cash locations. Airtm + Bridge across 150+ countries. Agent-initiated transfers arrive as local currency — no crypto knowledge needed on the receiving end.
MoneyGram · Airtm
L5
Stablecoin liquidity
Liquidity layer
PayPal PYUSD launched on Stellar. OpenZeppelin building secure Soroban contract templates. USDC volume on Stellar: ~$500M/month — real liquidity for agents to operate against.
PYUSD · USDC · OZ
L6
Agent infra momentum
Ecosystem
Wirex and Ultra Stellar shipping native Stellar payment infra on Soroban — positioned to onboard millions of users and autonomous financial agents alongside StellarClaw.
Wirex · Ultra Stellar
